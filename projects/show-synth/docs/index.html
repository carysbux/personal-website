Episode Emotion Analysis Dashboard

Sentiment analysis of Game of Thrones — emotional tone vs. episode ratings

Introduction

This dashboard explores the relationship between emotional tone in Game of Thrones episode synopses and various success metrics. Using sentiment analysis, we extract emotion scores from each episode's plot description and run linear regression against ratings and viewership data.

Goals

• Identify which emotions in episode synopses correlate with higher ratings (IMDb, Rotten Tomatoes, Metacritic).

• Understand how emotional tone relates to U.S. viewership and episode running time.

• Visualize the relative importance of each emotion across different target variables.

Issues

  • R² scores are modest; emotion scores alone explain a limited share of variance in ratings.
  • Small sample size (73 episodes) limits statistical power.
  • Synopsis text may not fully capture on-screen emotional content.

In the Future

  • Expand to dialogue or subtitle analysis for richer emotion signals.
  • Include season-level or character-level aggregations.
  • Test non-linear models or interaction terms.
